== Description ==

Starter Blog is a super fast powerful and versatile WordPress Gutenberg ready theme for the Bloggers. Clean and minimalist theme that allows your reader to focus on your content, build especially for daily or hobby bloggers who love to share their stuff, whether its about lifestyle, fashion, travel, entertainment, beauty or just a simple personal blog. Starter Blog theme best for single niche or multi niche blogging and SEO optimized. Bootstrap based fully RTL Supported. Extremely fast and easy to use. It's not just a theme â€“ It's a dream.

== Installation ==

= From within WordPress =
1. Visit "Appearance > Themes > Add New"
1. Search for "Starter Blog"
1. Install and activate

== Upgrade Notice ==
= Method 1 =
"Your site Address"/wp-admin/update-core.php

= Method 2 =
Go to "Appearance" > Themes >
You'll see the theme upgrade notice. Just click on the update.

== Frequently Asked Questions ==
= Is Starter Blog Free? =
Yes! Starter Blog is a free theme, and always will be.

= Does Starter Blog have a pro version? =
It does! Starter Blog has a premium plugin which extends the available options in the theme.

You can learn more about [here](https://fancywp.com/downloads/starter-blog-pro/).

= Where can I find documentation? =
Starter Blog has extensive documentation you can find [here](https://fancywp.com/docs/starter-blog/).

= Do you offer support? =
Definitely. We offer support for the free theme in the [WordPress.org forums](https://wordpress.org/support/theme/Starter Blog).

Premium customers have access to our very own [ticket](https://fancywp.com/contact).

We try to answer all questions - free or premium - within 24 hours.

= Where can I find the theme options? =
All of our options can be found in the Customizer in 'Appearance > Customize'.

= Does Starter Blog have any widget areas? =
Starter Blog has up to 6 widget areas which you can add widgets to in Appearance > Widgets.

= How can I make my site look like your screenshot? =
If you want to replicate the screenshot you see on WordPress.org, please refer to [this article](https://fancywp.com/docs/starter-blog/).
